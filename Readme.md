# API Librería - Flask

## Requisitos
- Python 3.11+
- (Opcional) Miniconda

## Instalación
1. Crear y activar entorno (conda o venv).
2. `pip install -r requirements.txt`
3. Crear la BD y migraciones (ver sección 'Migraciones').

## Ejecutar
`python app.py`  (o `flask run` si exportaste FLASK_APP=manage.py)

Swagger UI: http://127.0.0.1:5000/swagger  
OpenAPI JSON: http://127.0.0.1:5000/openapi.json
